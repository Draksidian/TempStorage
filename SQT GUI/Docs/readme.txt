===============================================================================	
Description	:
In world where trends change from year to year, updating GUI to keep pace with trends needs a special design. This project proposes an architecture with a working example
how Empower could be changed to meet the requirements.

===============================================================================	
Usage	:
Compile Empower in debug mode
Place OQ_backend directory in the root Empower source directory
Compile OQ_backend project in debug mode
run project to start HTTP server
test HTTP API with provided frontends ( html version only like IE )
